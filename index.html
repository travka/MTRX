Hotkeys

  • 1: toggle fullscreen
  • 2: decrease font size
  • 3: increase font size
  • 4: hue +
  • 5: hue -
  • 6: saturation +
  • 7: saturation -
  • 8: speed up falling
  • 9: slow down falling
  • [ : longer trails
  • ] : shorter trails
  • Backquote (`): toggle debug info

Additional Hotkeys

  • L: long trail
  • K: short trail
  • S: increase saturation
  • A: decrease saturation
